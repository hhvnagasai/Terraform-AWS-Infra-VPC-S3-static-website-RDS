# Terraform-AWS-Infra-VPC-S3-static-website-RDS

This project demonstrates the provisioning of a complete AWS infrastructure using Terraform with a modular and secure approach. It includes the creation of a custom VPC with public and private subnets, networking components such as Internet Gateway, NAT Gateway, and route tables, along with a static website hosted on Amazon S3 and a MySQL RDS instance deployed in private subnets.

The implementation follows Infrastructure as Code (IaC) principles, emphasizes security best practices by avoiding hardcoded credentials, and showcases real-world DevOps concepts such as modular design, resource isolation, and automated provisioning.
